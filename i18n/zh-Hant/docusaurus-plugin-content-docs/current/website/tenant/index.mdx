# 什麼是租戶？

訪客管理系統（VMS）允許租戶與大樓相關聯，並請求進入大樓。此外，租戶還可以管理其訪客的信息，包括訪客歷史。該系統允許租戶創建管理員和普通用戶帳戶，以設計各種申請任務的專屬審批流程。
