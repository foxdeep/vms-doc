# 描述

在"收件匣"，使用者可以接收與大樓相關的申請單。使用者可以點擊表格中顯示的申請單以查看更多詳細信息。在申請單中，使用者可以選擇“同意”或“退回至申請人”。

<BrowserWindow url={'https://vms.ces.myfiinet.com/bm/inbox'}>

![snapshot](/assets/bm/bm-inbox-ticket-zh-hant.png)

</BrowserWindow>

## 申請單詳情預覽

使用者可以通過在用戶偏好設置中添加申請單欄位（來自“表單模板”）來查看申請單的詳細信息。用戶偏好設置可以在窗口右上角的“頭像”中找到。
