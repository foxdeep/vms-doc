# 描述

“設置” 包含 “資訊”、“承攬商附件配置”、“承攬人員附件配置”、“功能設置”、“樓層管理” 三個分頁。

## 資訊

用戶可以上傳大樓照片，也可以修改地址和堡主。

## 功能設置

允許用戶打開或關閉“樓層管理”、“OWL FaceID 服務”和“承攬人員教育訓練”等常用功能。如果用戶需要額外的功能，他們可以聯繫管理員做自定義配置。

## 承攬商/ 承攬人員附件配置

“承攬商附件配置”和“承攬人員附件配置”標籤頁允許用戶配置上傳承攬人員和人員附件所需的欄位。每個附件配置需要以下欄位：

- 名稱
- 通知條件
  - 空: 指定當欄位為空時，是否應自動通知樓管。
- 保存後更新欄位: 每當另一個欄位更新時，相應地更新此欄位。

<BrowserWindow url={'https://vms.ces.myfiinet.com/bm/building/settings'}>

![snapshot](/assets/bm/bm-settings-attachment-config-zh-hant.png)

</BrowserWindow>

<Tabs>
  <TabItem value="type" label="附件类型" default>

管理員可以配置上傳承攬商和人員附件所需的欄位。每種附件類型都有其自己的配置選項。可用的類型如下：

- 文本
- 日期
- 文件：可以設置可接受的文件類型，如 `pdf`、`doc`、`xls` 等。
