# 描述

在我們的“大樓”中，我們有三個主要區塊。 “資訊”區顯示有關大樓的基本信息，包括二維碼。 “管理”區列出了管理小組。 最後下方提供各種功能的入口，包括“公告”、“承攬商”、“註冊單位”、“申請紀錄”、“ID 掃描器”、“進出紀錄”和“設置”。
