＃ 描述

**施工申請總表**為用戶提供了多種選項，用於過濾和搜索特定時間範圍內的大樓申請單。 用戶可以選擇今天、本週、本月、今年等時間範圍，也可以設置自定義範圍。 此外，用戶可以根據申請單是否為已完成或當前用戶簽核來過濾。搜索功能允許用戶通過輸入申請單名稱來查找。

此外，用戶可以將所有搜索結果輸出為 Excel 工作表，進而分析或統計。
