# 描述

“模板管理”包括三個主要功能：

- 同意書模板：用戶可創建一個同意書表單，此表單包含簽名欄位。
- 表單模板：用戶可創建各種類型的表單，例如一般、訪客、施工或駐廠。 這些表單用於收集申請進入大樓的申請人信息。 每個表單模板將在租戶介面將以申請單呈現。
- 申請任務模板：允許用戶將不同的表單模板綁定在一起，在租戶介面將以申請任務呈現。
