# VMS 是什麼?

**訪客管理系統（VMS）**是一種用於管理訪客的系統。
通常，任何<mark>不是全職員工</mark>的人都被視為**訪客**。
必須對進出建築物（工廠/辦公室）的訪客進行適當的安排和管理，以及參與任何活動。
提供了一個通用的辦公自動化平台，以有效地管理訪客進出的記錄。

市場上有少數通用的 VMS 系統，其中大多數是公司特定的註冊系統（或<mark>紙質系統</mark>）。
雲平台團隊旨在提供 SaaS 服務，建立一個適用於內部企業和一般商業辦公室的管理服務。

## 為什麼需要它

- 把握訪客流量
- 快速識別身份
- 無紙化和環保環境
- 管理活動記錄
- 確保訪客/員工的安全

### 優點

- 🚀 提高效率
  - 提供快速的訪客身份確認，加快進入速度，節省時間。
- ♻️ 環境友善
  - 數位化的流程減少紙張使用，符合 ESG（環境、社會、公司治理）要求。
- 📈 報告
  - 提供用戶可下載的報告數據。
- 🗄️ 安全
  - 除了數據安全外，快速的人員管理反應為用戶提供了更大的安全保障。
- 🔍 查詢
  - 用戶可以隨時查詢歷史數據。
- 📆 預約
  - 使用系統化和自動化的預約系統。
