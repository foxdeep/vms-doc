# 描述

在"收件匣"，使用者可以接收与大楼相关的申请单。使用者可以点击表格中显示的申请单以查看更多详细信息。在申请单中，使用者可以选择“同意”或“退回至申请人”。

<BrowserWindow url={'https://vms.ces.myfiinet.com/bm/inbox'}>

![snapshot](/assets/bm/bm-inbox-ticket-zh-hant.png)

</BrowserWindow>

## 申请单详情预览

使用者可以通过在用户偏好设置中添加申请单栏位（来自“表单模板”）来查看申请单的详细信息。用户偏好设置可以在窗口右上角的“头像”中找到。
