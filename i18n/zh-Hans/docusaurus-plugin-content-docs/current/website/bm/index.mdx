# 什么是楼管？

**楼管 (BM)** 是 VMS 平台的中心枢纽，旨在帮助物业经理有效地监督和运营其大楼。通过 BM，用户可以利用广泛的功能，包括：

- 承揽商
  - 承揽商和人员管理
  - 上传附件
  - 入场证二维码
  - 追踪许可证到期日期的状态
- 模板管理
  - 任务模板中的自定义表单
  - 表单模板中的自定义问题和签核人
- 用户管理
  - 创建用户/管理员角色
- 进/出记录
- 公告
  - 轻松向所有租户发布公告
  - 使用所见即所得的编辑器进行编辑
  - 状态控制
- ID 扫描器
  - ID 扫描器管理
  - 单一扫描器的进/出记录
