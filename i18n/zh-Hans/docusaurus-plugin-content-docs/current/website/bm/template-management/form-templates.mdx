# 描述

在“表单模板”，用户可以看到“一般”、“访客”、“施工”、“驻厂”四个分页，分别代表不同类型的表单模板。这些表单用于收集申请进入大楼的申请人信息。每个表单模板将在租户介面中以申请单呈现。

在每种类型的表单模板表中，用户可以看到以下列：

- “创建类型”：该栏表示表单模板是自订还是预设的模板。如果是“预设”模板，用户不能修改表单内容或删除； 他们只能修改表单模板的名称。
- “发布状态”：该栏表示表单模板是否已经发布给租户。
- “名称”
- “创建时间”

## 创建

要创建表单模板，需要以下栏位：

- 复制模板来源
- 名称

## 编辑

创建表单模板后，用户可以点击“更多”按钮中选单，以修改表单模板的信息、内容、签核管理和相关附件。

### 资讯

当用户点击“更多”选单中的“编辑资讯”时，会在对话框中显示相关表单模板的可编辑信息。所有显示的字段都可以编辑。

- “发起者”：指将发起申请单的身分。
- “结束者”：指将结束申请单的身分。

<BrowserWindow url={'https://vms.ces.myfiinet.com/bm/template-management/form-templates'}>

![snapshot](/assets/bm/bm-edit-form-template-zh-hant.png)

</BrowserWindow>

点击“更多”选单中的“表单设计”后，会向上滑动一个全屏对话框，呈现三个分页：“表单架构”、“签核管理”和“附件管理”。

#### 表单架构

<Tabs>
  <TabItem value="questionType" label="问题类型" default>
  在“表单架构”分页中，用户可以使用各种问题类型修改表单架构，包括：

- 文字
  - 简答：一个“单行”文字栏位
  - 详答：一个“多行”文字栏位
- 选择
  - 核取方块：包含多个选项的复选框组，其中可以选择“多个”选项作为答案。
  - 选择题：包含多个选项的单选组，其中可以选择“一个”选项作为答案。
  - 下拉式选单：包含多个选项的下拉列表，其中可以选择“一个/多个”选项作为答案。
  - 自动补全选单：包含多个选项的下拉列表，其中可以选择“一个/多个”选项作为答案。用户还可以在文字栏位中输入以过滤选项。
  - 🚧 图片选择器
- 是非题：以“是/否”作为答案的开关。
- 🚧 排序
- 档案上传：返回“档案”作为答案。
  - 管理员可以限制档案类型、档案数量和档案大小。
- 日期时间
  - 日期选择器：返回“日期”作为答案的选择器（例如 2023-04-07）。
  - 时间选择器：返回“时间”作为答案的选择器（例如 08:00）。
  - 日期时间选择器：返回“日期时间”作为答案的选择器（例如 2023-04-07 08:00）。
  - 日期范围选择器：返回“日期范围”作为答案的选择器（例如 2023-04-07 ~ 2023-04-08）。
- 图片编辑器：允许用户在图像上添加注释的编辑器，例如绘图或文字。

:::note
符号 🚧 表示特定问题类型当前正在开发中，尚未发布。
:::

  </TabItem>

<TabItem value="attribute" label="属性">

每种问题类型的共同属性包括：

- 问题：问题的名称
- 必填：问题是否必填

特定问题类型的独特属性包括：

- 简答
  - 文字类型（普通文字/电话）
    - 普通文字：一个“纯文字”字段
    - 电话：验证给定国家/地区格式的“电话号码”字段。
- 核取方块
  - 最小/最大选择：用户可以选择的选项数量
- 日期范围选择器
  - 最短/最长天数：用户可以选择的日期范围数
  - 开始时间/结束时间：日期的开始时间/结束时间
- 自动补全选单
  - API：一个开关，可以使用来自系统的数据，例如“楼层”、“承揽商”和“承揽人员”。

#### 签核管理

在发布表单模板之前，设置签核流程非常重要。用户可以轻松配置在关闭申请单之前应有多少人签核。

如果身分是租户，则申请人将在申请单创建时指定签核人。若是楼管的身分，则需要选择一个用户作为该阶段的签核人。另外，用户需要为每个阶段分配合适权限的签核人。
