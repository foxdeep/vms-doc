# 描述

“申请任务模板”包含四个分页，名称分别为“一般”、“访客”、“施工”、“驻厂”。这些分页代表不同类型的任务模板，在租户介面中以申请任务呈现。

每种类型的申请任务模板表包括以下列：

- 创建类型：该栏表示任务模板是自订还是预设的模板。如果是系统默认模板，用户只能修改表单模板的名称和流程。
- 名称：此列显示任务模板的名称。
- 创建时间：此栏显示创建任务模板的日期和时间。

## 编辑

目前，我们只发布了预设的任务模板。用户可以通过单击表格最后一列中的“更多”选单并选择所需的选项来修改任务模板的名称和流程。
