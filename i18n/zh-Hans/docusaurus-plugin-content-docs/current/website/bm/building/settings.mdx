# 描述

“设置”页面包括三个标签页，分别是“资讯”、“承揽商附件配置”、“承揽人员附件配置”、“功能设置”和“楼层管理”。

## 资讯

用户可以上传大楼照片，也可以修改地址和堡主。

## 功能设置

"功能设置"标签页可以让用户轻松开启或关闭常用功能，如"楼层平面图"、"OWL 人脸识别服务"和"管制密码"。此外，用户还可以调整访客应用程序 Upass 的"弹性打卡时间"。如果您需要更多功能，请随时联系管理员进行个性化定制。

## 承揽商/ 承揽人员附件配置

“承揽商附件配置”和“承揽人员附件配置”标签页允许用户配置上传承揽人员和人员附件所需的栏位。每个附件配置需要以下栏位：

- 名称
- 通知条件
  - 空: 指定当栏位为空时，是否应自动通知楼管。
- 保存后更新栏位: 每当另一个栏位更新时，相应地更新此栏位。

- 文本
- 日期
- 文件：可以设置可接受的文件类型，如 `pdf`、`doc`、`xls` 等。
