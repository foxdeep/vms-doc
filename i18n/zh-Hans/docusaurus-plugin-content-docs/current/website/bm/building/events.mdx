# 描述

在“事件”页面上，用户可以查看大楼的进出记录表。他们能够根据特定条件或访问者名称过滤事件。

## 编辑

当用户点击“更多”菜单中的“编辑”时，相关事件的可编辑信息将显示在对话框中。所有显示的栏位都可以编辑。
