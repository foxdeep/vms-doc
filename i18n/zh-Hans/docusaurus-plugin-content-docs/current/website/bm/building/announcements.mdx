# 描述

“公告”允许用户以网格查看与大楼相关的公告。用户可以按状态过滤公告，并按发布时间对公告进行排序。用户还可以在此页面上创建、编辑或删除公告。

## 创建

要创建公告，需要以下栏位：

- 标题
- 内容：此栏位为 WYSIWYG 编辑器，允许用户复制和粘贴内容，同时保持其原始格式。

完成所有必填栏位后，用户可以选择`储存`或`发布`公告。

- 储存：已储存，但不会发布给租户。
- 发布：已储存并发布给租户。

## 编辑

当用户点击“更多”选单中的“编辑”时，会在对话框中显示相关公告的可编辑信息。所有显示的栏位都可以编辑。

<BrowserWindow url={'https://vms.ces.myfiinet.com/bm/building/announcements'}>

![snapshot](/assets/bm/bm-edit-announcements-zh-hant.png)

</BrowserWindow>

## 删除

当用户点击“更多”选单中的“删除”时，会弹出一个警告对话框，再次确认并提醒用户删除操作不可逆。
