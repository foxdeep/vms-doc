# 描述

在“注册单位”页面上，用户可以查看大楼所有已注册单位。成为注册单位意味着该单位可以申请进入大楼，查看承揽商和人员名单，并接收已发布的公告。

## 邀请

要邀请单位，请按照下列步骤操作：

- 点击页面右下角的“邀请”按钮。
- 将出现一个对话框，其中包含以下栏位：
  - 名称
  - 使用者名称
  - 电子邮件

## 删除

当用户点击“更多”选单中的“删除”时，会弹出一个警告对话框，再次确认并提醒用户删除操作不可逆。
