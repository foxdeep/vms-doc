# 描述

在我们的“大楼”中，我们有三个主要区块。 “资讯”区显示有关大楼的基本信息，包括二维码。 “管理”区列出了管理小组。最后下方提供各种功能的入口，包括“公告”、“承揽商”、“注册单位”、“申请纪录”、“ID 扫描器”、“进出纪录”和“设置”。
