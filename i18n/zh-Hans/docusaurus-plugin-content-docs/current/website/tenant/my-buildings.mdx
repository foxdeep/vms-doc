# 描述

**我的大楼**允许用户检视已注册大楼的最新公告，与申请其他大楼的注册单位。

## 大楼资讯

用户点击**进入大楼**按钮后，他们将能够访问与大楼相关的所有相关信息，例如公告、承揽商和承揽人员详细信息。
