# 什么是租户？

访客管理系统（VMS）允许租户与大楼相关联，并请求进入大楼。此外，租户还可以管理其访客的信息，包括访客历史。该系统允许租户创建管理员和普通用户帐户，以设计各种申请任务的专属审批流程。
