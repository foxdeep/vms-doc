# VMS 是什么?

**访客管理系统（VMS）**是一种用于管理访客的系统。
通常，任何<mark>不是全职员工</mark>的人都被视为**访客**。
必须对进出建筑物（工厂/办公室）的访客进行适当的安排和管理，以及参与任何活动。
提供了一个通用的办公自动化平台，以有效地管理访客进出的记录。

市场上有少数通用的 VMS 系统，其中大多数是公司特定的注册系统（或<mark>纸质系统</mark>）。
云平台团队旨在提供 SaaS 服务，建立一个适用于内部企业和一般商业办公室的管理服务。

## 为什么需要它

- 把握访客流量
- 快速识别身份
- 无纸化和环保环境
- 管理活动记录
- 确保访客/员工的安全

### 优点

- 🚀 提高效率
  - 提供快速的访客身份确认，加快进入速度，节省时间。
- ♻️ 环境友善
  - 数位化的流程减少纸张使用，符合 ESG（环境、社会、公司治理）要求。
- 📈 报告
  - 提供用户可下载的报告数据。
- 🗄️ 安全
  - 除了数据安全外，快速的人员管理反应为用户提供了更大的安全保障。
- 🔍 查询
  - 用户可以随时查询历史数据。
- 📆 预约
  - 使用系统化和自动化的预约系统。
