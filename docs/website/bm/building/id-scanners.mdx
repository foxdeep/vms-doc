# Description

On the "ID Scanners" page, users can view a table of available ID scanners for the building. Users can create ID scanners for the building using the provided license key, which has a set number of available and expiration date. Under each license key, users can see a table of available ID scanners, including their ID, name, and last updated time. The "More" button in the last column provides options to edit or delete the ID scanner. By clicking the row of scanners, users can view the access events of the scanner.

:::note
If you require additional license keys, please contact us.
:::

## Create

To create a scanner, the following field is required:

- Name

The following field is optional:

- Owl buddy ID (only appears if the building has an OWL module)

## Edit

When the user clicks on "Edit" in the "More" menu, the editable information of the relevant scanner will be displayed in a dialog box. All of the fields presented can be edited.

<BrowserWindow url={'https://vms.ces.myfiinet.com/bm/building/scanners'}>

![snapshot](/assets/bm/bm-edit-id-scanner-zh-hant.png)

</BrowserWindow>

## Delete

When the user clicks on "Delete" in the "More" menu, a warning dialog box will be displayed to double-check and alert the user that the delete action is irreversible.

## Events

By selecting the row of scanners, users can access the log of access events for that particular scanner. Users have the option to filter the events by date range and personnel name. At the conclusion of each row, users can click on the “View” button to retrieve the ticket associated with the event.
