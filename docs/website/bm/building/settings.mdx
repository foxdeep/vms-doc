# Description

The "Settings" page comprises three tabs, including "Info", "Contractor Attachment Config", "Personnel Attachment Config", "Function Settings", and "Floor Planner".

## Info

Within the "Info" tab, users can upload a building image, as well as modify the address and Main PIC details with the approval of the administrators.

## Function Settings

The "Function Settings" tab allows users to toggle the common functions such as "Floor Planner", "OWL FaceID Service", and "Personnel Orientation" on or off. In case users require additional functions, they can contact the administrator to customize the configuration.

## Contractor/ Personnel Attachment Config

The "Contractor Attachment Config" and "Personnel Attachment Config" tabs allow users to configure the required fields for uploading attachments for contractors and personnel. Each attachment config requires the following fields:

- Name
- Notify condition
  - Empty: Specifies whether the attachment should be automatically notified to BM when the field is empty.
- Update the field after saving: whenever another field is updated, update this field accordingly.

- Text
- Date
- File: can set acceptable file types such as `pdf`, `doc`, `xls`, etc.

## Floor Planner

In the "Floor Planner" tab, users can add a new floor, upload its image, and set up the permit number as well as the list of tenants located on that floor.
